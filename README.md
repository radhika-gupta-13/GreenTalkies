## 🌱 Overview

**GreenTalkies** is a smart gardening ecosystem built with Flutter that helps users grow plants, learn gardening, and connect with a sustainable community 🌍🌿

It combines:
- 🌱 Plant intelligence  
- 🤖 AI gardening assistant (**Greenie**)  
- 🌦️ Weather-based care insights  
- 🧑‍🌾 Community discussions  
- 🛒 Eco-friendly marketplace (**Bud & Basket**)  

> 🌿 *Grow smarter. Live greener. Connect better.*
